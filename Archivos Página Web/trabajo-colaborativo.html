Proceso de aprendizaje

Reflexión grupal — trabajo colaborativo

Cómo se organizó el equipo para desarrollar el proyecto: distribución de roles, herramientas de coordinación y lecciones del trabajo conjunto.

Organización

Roles y responsabilidades

El trabajo se distribuyó por módulos técnicos según las fortalezas de cada integrante, con responsables definidos para el proceso, la simulación, el control y la evaluación económica.

Herramientas

Coordinación del equipo

GitHub como repositorio central del proyecto, mensajería para la coordinación diaria y sesiones de trabajo conjunto para las integraciones entre módulos.

Lecciones

Lo que aprendimos del trabajo conjunto

La integración entre módulos exige acuerdos tempranos: definir interfaces claras (señales, formatos, supuestos) evitó retrabajos en las etapas finales del proyecto.
